## 📖 About The Project
 
**BakeHaven** is an e-commerce web application for an artisanal bakery. It allows customers to browse fresh products, add items to a cart, and place orders — all from their browser, with no long queues.
 
The project was built as a graduation/team project to demonstrate a real-world frontend solution to a common problem: the chaos and crowds at traditional bakeries. Instead of waiting in line, customers can now shop online, pre-order, and pick up their bread with ease.
 
> *"Discover the warmth of artisanal breads and decadent pastries made with organic ingredients and a dash of love."*
 
---
 
## ✨ Features
 
- 🏠 **Home Page** — Hero section, category showcase, and daily deals
- 🛍️ **Products Page** — Browse and filter by category (Bread, Pastry, Cake, Cookie)
- 🛒 **Shopping Cart** — Add/remove items, live cart count in the navbar
- 🔐 **Authentication** — Login / Logout functionality
- 📦 **Order System** — Organized digital orders (no paper, no chaos)
- 💬 **Feedback Form** — Customers can submit feedback directly from the site

## 🚀 Getting Started
 
### View Live
👉 [https://eslam-lc.github.io/E-Commerce-Bakery/index.html](https://eslam-lc.github.io/E-Commerce-Bakery/index.html)
 
### Run Locally
 
```bash
# 1. Clone the repository
git clone https://github.com/Eslam-LC/E-Commerce-Bakery.git
 
# 2. Navigate to the project folder
cd E-Commerce-Bakery
 
# 3. Open index.html in your browser
#    (No build tools or dependencies needed!)
open index.html
```
